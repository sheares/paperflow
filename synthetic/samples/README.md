# scanned-doc samples

A small set of image documents (PNG/JPEG) used to demo the Real-pile upload flow. These are NOT part of any pile's ground truth; drag them into the "Your pile" tab in the UI to see Gemma extract the fields live and Presidio redact identifiers.

- `nric_scan_chloe.png` — Singapore NRIC-shaped ID card photo
- `utility_bill_priya.jpg` — scanned utility bill with a rubber-stamp receipt marker
- `employment_letter_priya.jpg` — signed employment verification letter
